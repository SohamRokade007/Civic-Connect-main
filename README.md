# 🏛️ Civic Issue Reporting & Resolution App

A modern, accessible web application that enables residents to report location-based civic issues and track their resolution status in real-time.

## ✨ Features

### For Residents
- 📍 **Location-based Issue Reporting** - Report issues with precise location data
- 📱 **Mobile-First Design** - Optimized for field reporting on smartphones
- 🔍 **Real-time Tracking** - Monitor issue status from report to resolution
- 🏷️ **Smart Categorization** - Organized by issue type (potholes, streetlights, etc.)
- 📊 **Public Dashboard** - Transparent view of all community issues

### For Municipal Authorities
- 🛠️ **Issue Management** - Assign, update, and resolve reported issues
- 📈 **Analytics Dashboard** - Track resolution metrics and performance
- 👥 **Role-based Access** - Secure administrative controls
- 🔄 **Status Updates** - Real-time communication with residents

## 🚀 Quick Start

### Prerequisites
- Node.js (version 16 or higher)
- npm or yarn package manager

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/YOUR_USERNAME/civic-issue-reporting-app.git
   cd civic-issue-reporting-app
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:5173`
